OIDC Kubeconfig

Download a kubeconfig that authenticates via OIDC using kubectl-oidc-login.

Setup instructions:

1. Install kubectl-oidc-login:
kubectl krew install oidc-login

2. Download and save the kubeconfig above

3. Use it:
export KUBECONFIG=~/Downloads/kubeconfig--oidc.yaml
kubectl get nodes

4. A browser window will open for Keycloak authentication on first use.

RBAC group mappings (prefix oidc:):
• platform-admins → cluster-admin
• k8s-admins → cluster-admin
• k8s-operators → k8s-operator (read-only)
